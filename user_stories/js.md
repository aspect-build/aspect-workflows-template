## Developer tools

The Bazel-managed version of pnpm is on the PATH thanks to direnv:

~~~sh
pnpm list
~~~

We can use Yeoman to scaffold out a library and add its dependencies:

~~~sh
pnpm add -w generator-bazel-fastify-route
yo bazel-fastify-route
~~~
